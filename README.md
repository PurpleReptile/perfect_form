# Perfect Form
Небольшая справочная информация о проекте.

* структура проекта
* data-атриббуты 

### Структура проекта
* php 
    * PerfectForm.php
    * include.php
* js
    * PerfectForm.js
 * index.html

### Data - атрибуты

1. data-pf-open 
2. data-pf-name="NameForm"
3. data-pf-place="NameForm"

где NameForm - название вызываемой формы. 

> ВАЖНО: id формы должен совпадать с именем, которое указано 
в data-атрибутах.
